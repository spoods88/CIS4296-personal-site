Co-op Approval Automated System

For current CIS students that seek to apply for the Co-op course. The automated Co-op approval system will allow for a more seamless application process for the student, job supervisor, and faculty involved. The automated Co-op Approval system will streamline and improve the process by providing an online form, defined automated workflow, and email notifications. The automation will allow for the status of the application to be viewable to both the student as well as the faculty involved in the approval/denial review. Unlike in the current system in which the students have to manually manage each form and are solely responsible for the completion of the process.
